### 👨‍💻 О себе

Python-разработчик с опытом разработки на Python от 2 лет. Участвую в полном цикле создания backend-части для веб-приложений: от проектирования API и БД до релиза и развертывания на продакшен-сервере.

**Ключевые навыки:**
- Разработка REST API и бизнес-логики на Django, FastAPI, Flask
- Проектирование и оптимизация баз данных (PostgreSQL, Redis)
- Асинхронное программирование (asyncio, Celery, RabbitMQ)
- Docker, Linux, CI/CD
- Чистый код, тестирование (Pytest), code review
